# Equipment Journal

Telegram bot for voice logging of security rounds and equipment handover.

## Stack
- Frontend: GitHub Pages (HTML/JS)
- Backend: Cloudflare Workers
- Bot: Telegram Bot API
- LLM: Google Gemini API (structured JSON output)
- Database: Cloudflare D1 (SQLite)
